Initialising local repository
git status - It provides the overall health and visibility of the present repository.
git init - It initializes the git in your folder.
ls -a - It shows hidden files if any.
git add filename - It adds that file to the staging area.
git config --global-l - It shows the name and email of the local repository.
git add . - It adds all the files to the stage.
git commit -m - It is used to commit the files by developers.
git log - It gives information about commit with developer's name, email, time and date.
git diff - It shows the changes in file if any made. This command helps you getting the difference between last committed file and present file.
